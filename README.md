## 需求

- **Hearts of Iron 4 1.18.2**（"Peace For Our Time" DLC 周期）
- 原版游戏文件（用于 grep 学习语法，默认路径 `D:\steam\steamapps\common\Hearts of Iron IV`）
- 建议安装 **VS Code + CWTools 扩展**（语法高亮、错误检查）
- 无需外部 MCP 服务器——直接读取本地游戏文件

## 设计原则

- **模板优先**: 有模板直接用，无模板 grep 原版学习后存为新模板
- **测试版先行**: 所有 Mod 先生成测试版，验证通过后再正式化
- **贴图默认空白**: AI 无法生成 .dds/.tga 贴图，所有图形字段使用占位符，并引导用户补充
- **代码原创性**: 强制禁止抄袭其他 Mod 代码，只允许参考原版文件和学习技术思路
- **国策坐标引导**: AI 无法预览布局，生成后引导用户用 `reloadfocus` 实时调整坐标
- **Steam 查重**: 生成 Mod 前搜索 Workshop 已有内容，帮用户找到差异化方向
- **中国内容合规**: 涉及中国相关内容时自动提醒法律风险和免责声明
